Overview

这是我的个人网站 UI toolkit，除了少数组件，大多数 css 都不包含任何 class 选择器，适用于各类静态网站生成器所生成的网页。

Usage

引入下面的 style 标签到你的页面:

  <link rel="stylesheet" href="http://www.songofcode.com/socss/assets/toolkit/styles/toolkit.css">
